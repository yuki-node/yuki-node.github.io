# yuki-node.github.io

https://yuki-node.github.io/

## Quick Start

```shell
brew install hugo
hugo server
```

## Write Blog

```shell
# create a draft
sh create-post.sh
```

## Deploy Blog

Auto deploy by github action, the config workflow is in the  `.github/workflows/hugo.yml`

## How to use images

1. Hugo get the images from `static` directory;
2. use the images: `![xxx](/images/xxx/xxx.png)`
